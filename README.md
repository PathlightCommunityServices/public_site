# Pathlight Community Services (mipath.org)
Welcome to the official repository for Pathlight Community Services. This repository hosts the source code and static files for our main website, mipath.org, served via GitHub Pages.

## About Pathlight Community Services
Pathlight Community Services is an Adult Foster Care 401(c)3 non-profit organization that provides care to vulnerable adults by providing housing for hundreds of clients across 5 counties in Southeast Michigan. 

Our website serves as the primary hub for:

- Providing contact information
- Providing link to the client referral page
- Providing link to the staff portal
